BOM：brower object model 浏览器对象模型，主要处理浏览器窗口和框架。

bom 包含了window， window 包含document，bom包含了dom

  • window 浏览器窗口
  • navigator 浏览器信息
  • history 历史记录
  • location 当前url信息
  • screen 显示

window

api：confirm 弹出确认框 close 关闭窗口 open(有参数)打开窗口 prompt 提示用户的输入框 scrollby每次滚动多少 scrollto 滚到到啥位置

属性： closed当前窗口是否被关闭 innerHeight innerWidth 显示文档区域宽高 pageXoffset pageYoffset

navigator

api：confirm 弹出确认框 close 关闭窗口 open(有参数)打开窗口 prompt 提示用户的输入框 scrollby每次滚动多少 scrollto 滚到到啥位置

属性：cookieEnabled 是否启用cookie值 返回boolean值 online 是否处于联网状态 userAgent 返回发送服务器的头部值

history

api：back forward go

属性：length 历史记录长度

location

api：reload 重新加载 assign 加载新的文档 replace 用新的文档替换当前文档

属性：hash url#后的值 host 主机名和端口号 href 完整的url search ？+后面的内容 protocol 协议名

screen

api：reload 重新加载 assign 加载新的文档 replace 用新的文档替换当前文档

属性：hash url#后的值 host 主机名和端口号 href 完整的url search ？+后面的内容 protocol 协议名
